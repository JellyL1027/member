登入界面(login.php): 一開始的界面， 可以在這裏登入或加入會員
加入會員(createAcc.php): 加入會員的界面， submit後會到
判斷帳號(checkCreateAcc.php): 判斷帳號是否成功注冊
會員界面(member.php): 登入成功後的界面， 可以瀏覽資料、修改資料、變更密碼、登出
瀏覽資料(profile.php): 會員可以看到個人資料
修改資料(changeProfile.php): 會員可以修改個人資料
變更密碼(changePW.php): 變更密碼， 輸入一次舊密碼和兩次新密碼
登出(logout.php): 返回到登入界面， 並unset Session
connect.php: 連缐資料庫
